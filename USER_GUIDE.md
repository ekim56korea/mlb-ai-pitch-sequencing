# Pitch Commander Pro User Guide

## 1. 매치업 설정 (Sidebar)
1. **선수 입력:** 분석하고 싶은 투수와 타자의 이름(영어)을 입력합니다. (예: `Kim Ha-Seong`)
2. **데이터 로드:** `📥 데이터 로드` 버튼을 누르면 서버가 최신 데이터를 가져옵니다. 로딩이 완료되면 "Success" 메시지가 뜹니다.
3. **환경 설정:** 경기장의 기온, 고도, 습도를 설정합니다. (쿠어스필드 시뮬레이션 시 고도를 5200ft로 설정하세요.)

## 2. 분석 및 시뮬레이션 (Analysis Tab)
* **자동 세팅:** 투수가 로드되면, 그 투수가 던지는 구종과 평균 구속이 자동으로 슬라이더에 세팅됩니다.
* **미세 조정:** 슬라이더를 움직여 "구속을 2mph 올렸을 때" 또는 "회전수를 높였을 때"의 변화를 실험합니다.
* **실행:** `🚀 시뮬레이션 실행` 버튼을 누르면 다음 결과가 나옵니다.
    * **Stuff+:** 구위 점수 (100점 평균).
    * **xRV:** 기대 득점 가치 (낮을수록 좋음).
    * **3D View:** 공의 궤적과 스트라이크 존 통과 위치.

## 3. 전략 수립 (Tactical Command)
1. **상황 입력:** 화면 상단에서 이닝, 볼카운트, 주자 상황을 설정합니다.
2. **AI 추천:** `🧠 전략 분석` 버튼을 누릅니다.
3. **결과 확인:** AI가 현재 상황에서 가장 확률 높은 구종과 코스, 그리고 그 이유(Reasoning)를 알려줍니다.
